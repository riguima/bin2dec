# Bin2Dec

Aplicação CLI que converte números binários para decimal, feito do reositório [app ideas](https://github.com/florinpop17/app-ideas)

## Instalação

Clone o repositório

```
$ git clone https://github.com/riguima/bin2dec
```

```
$ cd bin2dec
```

Compile

```
$ make
```

```
$ sudo make install
```

## Uso

```
$ bin2dec 1111
Binário: 1111
Decimal: 15
```

```
$ bin2dec 101010
Binário: 101010
Decimal: 42
```
